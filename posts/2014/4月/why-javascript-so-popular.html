Why javascript so popular

Posted on 周一 07 四月 2014 in web

Many peoples think javascript will be the most popular program language in the near future. Why? In my opinion, because the web will dominate the PC, smartphone and tablet. Which code can be coded once and run in almost every platform. The first response of everyone maybe the java. However, I would like to say javascript.

Why javascript? I think everyone need a web browser instead of a JAVA VIRTUAL MACHINE. Javascript can run in almost every modern browser. If you want to run JAVA or code JAVA program, you have to install JVM, JDK and so many other things to help you accomplish this task. Let's look in javascript. ALL you need is just a web browser and a text editor. So simple so easy. And because the javascript compiler(maybe a javascript interpreter) is embedded into browser, the java compiler not in browser cause this situation. If the java compiler is embedded into browser, maybe the javascripts' site will replace by java.

And you can use javascript to do almost every thing that tranditional program language do. They can connect to Database, they can make desktop application if they have compiler into OS which help you to make an .exe file in window platform and they can build web site and web application which is the main feature of it.
