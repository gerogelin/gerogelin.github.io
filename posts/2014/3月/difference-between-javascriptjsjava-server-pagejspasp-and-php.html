difference between JavaScript(JS),Java Server Page(JSP),ASP and PHP

Posted on 2014-03-20(周四) in web

JS is a script in web file, it download with html file in plain text. It is executed in client browser.

However, JSP, ASP and PHP are server technologies that run in server to create a html file depended on users' inquery.For example, when the user ask for weather data today, these technologies create a html file with data that represent weather today. When the user ask for weather data yesterday, these technologies create a html file with data that represent weather yesterday. IN SUMMARY, these three technologies are used to create html actively.

Then, the html will be send to browser, maybe with .js file(javascript file), the browser parser the html(also js file) and print it to screen.
